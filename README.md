# InterleavePDF

A static web app for merging two PDF files by alternating pages (A-B-A-B or B-A-B-A), fully locally in the browser.

## GitHub Pages

- https://artifexet.github.io/InterleavePDF/

## Project Structure

- `index.html` - application interface,
- `styles.css` - styling and responsiveness,
- `app.js` - client-side PDF merge logic,
- `vendor/pdf-lib.min.js` - local PDF processing library.

## Run Locally

Open `index.html` in your browser.
